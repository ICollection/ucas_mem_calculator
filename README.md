# 国科大工程管理课程计算器
实现部分课程中数值计算的功能，显示中间步骤和最终计算结果，方便核对数据。

## 软件开发环境
- Node.js（v22.11.0）
- Vue（v3.5.13）
- Element-Plus（v2.9.3）
- VSCode（v1.95.3）
